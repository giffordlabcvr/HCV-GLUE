Background

The emergence of clinically-relevant hepatitis C virus (HCV) resistance to direct-acting antiviral drugs in parallel with advances in next-generation sequencing technology means there is a need for computational resources that allow clinicians to make optimal treatment decisions based on analysis of virus sequence data. These resources will also be an important component of efforts to study and monitor HCV genetic variation, and to evaluate its impact on epidemiology, disease course and treatment outcome.

HCV-GLUE: Sequence Data Resource

HCV-GLUE is a bioinformatic resource for the curation and analysis of HCV sequence data. The resource can be used either via a web site or via an offline version aimed at more experienced bioinformaticians.

The web site includes:

  • A curated database of NCBI HCV sequences, with metadata.
  • A curated database of HCV clades (genotypes, subtypes) and reference sequences.
  • Curated multiple-sequence alignments for all the NCBI sequences assigned to each clade.
  • A curated database of amino-acid substitutions associated with drug resistance including their frequencies in different clades.
  • An analysis tool supporting genotyping and drug resistance analysis of submitted consensus (FASTA) sequences.

The offline version of the resource includes all the functionality available in the web site plus:

  • A rich set of bioinformatic functions for exploring the dataset.
  • Some basic tools for the analysis of NGS sequences (SAM/BAM files).
For more information about obtaining and using the offline HCV-GLUE resource, please contact Josh Singer.

Data Sources

Data relating to structural, evolutionary, epidemiological and phenotypic aspects of HCV were obtained from various sources.
  • HCV clade definitions and reference sequences were obtained from the ICTV HCV classification resource, updated in May 2015.
  • HCV virus sequences and their metadata were obtained from the NCBI Nucleotide database.
  • Drug resistance-associated substitutions were obtained from a number of recent surveys of in vitro experimental results.
The various datasets are integrated together in GLUE, a data-centric software framework for capturing, developing and utilising nucleotide-level virus evolutionary hypotheses.
