GLUE project

The HCV-GLUE web application is based on a linked dataset, packaged as a GLUE project. GLUE projects can be loaded into the GLUE engine which may be installed on a private computer. In this way, the HCV-GLUE functionality can be used for example to store private sequence data, and extended, for example to add custom metadata or new analysis functions.

Download GLUE project builds

The GLUE project comprises two layers, the core HCV-GLUE project, and the NCBI-HCV-GLUE extension. The core HCV-GLUE project contains the reference sequences and phylogenetic definitions along with RAS and epitope information. The NCBI-HCV-GLUE extension adds the HCV sequences derived from the NCBI nucleotide database and their metadata, the set of atypical substitutions and RAS frequency data.
