GLUE project

The HCV-GLUE web application is based on a linked dataset, packaged as a GLUE project. GLUE projects can be loaded into the GLUE engine which may be installed on a private computer. In this way, the HCV-GLUE functionality can be used for example to store private sequence data, and extended, for example to add custom metadata or new analysis functions.

Download GLUE project builds

Two project builds are available. The build including the NCBI-HCV-GLUE extension is larger but includes the HCV sequences derived from the NCBI nucleotide database and their metadata, the set of atypical substitutions and RAS frequency data.

HCV-GLUE core project + HCV-DRUG-RESISTANCE extension Download hcv_glue.sql.gz
HCV-GLUE core project + HCV-DRUG-RESISTANCE extension + NCBI-HCV-GLUE extension Download ncbi_hcv_glue.sql.gz
