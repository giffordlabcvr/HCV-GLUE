HCV-GLUE

This is the project page for HCV-GLUE, a GLUE project for Hepatitis C virus (HCV).

GLUE

GLUE is a software package for developing nucleotide-level evolutionary hypotheses for virus and endogenous viral element (EVE) genomes. It enables the dissemination of these hypotheses on the web and the utilisation of the hypotheses in wider software systems.

Included in the HCV-GLUE project

A complete genome alignment obtained from Smith et al (2014) was used to define reference sequences HCV genotypes.

The sequences of 3,227 non-redundant HCV isolates were obtained from GenBank and phylogenetically grouped into genotypes. These sequences were then aligned to genotype-specific reference sequences, and the resulting alignments were used to define common polymorphisms within each genotype.

This GLUE project also includes the GLUE framework implementation of a graphical web-based user interface (HCV-GLUE) that uses GLUE to produce interactive reports (Figure 1). Besides highlighting known DRMs, GLUE reports allow users to discriminate unusual mutations from common polymorphisms, and to investigate the frequency of specific mutations in reference datasets. Reports can be interactively configured to display information on a genotype or subtype-specific basis.
